#분류 모델 평가 ( 0,1인지 예측하는 것 / 실제 값도 0,1이고 예측 값도 0,1 이다. / 하지만 0을1로 예측하거나 1을 0으로 예측할 수 있음 ) -> 정확도를 높여라!

#회귀 모델 평가 ( 정확한 값을 예측하기 어려움 / 예측과 실제 값에는 오차가 존재 ) -> 오차를 줄여라! 

#y : 실제값 ( 목표 값 / 오차는 이 값과 예측 값의 차이 )
#yhat : 예측값 ( 머신러닝 알고리즘으로 새롭게 예측한 값 / 최소한 평균값 보다는 좋아야 함)
#ybar : 평균값 ( 이미 알고 있는, 기존에 예측한 값)

#오차는 하나의 값으로 ( 평균을 내면됨 ) -> 절대값의 합 / 제곱의 합
#Sum Squared Error : 제곱의 합 -> 18 
#Mean sSE  -> 제곱의 합의 평균 -> 4.5
#Root MSE -> 저곱의 합의 평균에 루트 -> 2.12

#Mean Absolute Error : 절대값의 합의 평균 -> 2
#Mean Absolute Percentage Error : 절대값의 합에 * 100 의 평균

#SST : Sum Squared Total : 전체 오차 
#SSR : Sum Squared Regression : 전체 오차 중에서 회귀식이 잡아낸 오차
#SSE : Sum Squared Error : 전체 오차 중에서 회귀식이 여전히 잡아내지 못한 오차
** SST = SSE + SSR **

# 결정계수 R^2 (R-Squared)
- MSE로 여전히 설명이 부족한 부분이 있다. ( 성능이 확실히 와 닿지 않음 )
- 모델 성능을 잘 해석하기 위해서 만든 MSE의 표준화된 버전이 결정 계수
- 전체 오차 중에서 회귀식이 잡아낸 오차 비율 ( 일반적으로 0~1 사이 )
- R^2 = 1이면 MSE = 0, 즉 모델이 데이터를 완벽하게 학습한 것

** R^2 = SSR/SST = 1 - ( SSE/SST) ** ** R^2는 클 수록 좋음 / MSE, RMSE , MAE, MAPE는 오류이므로 작을 수록 좋음 **
** 단순히 오차 절대값, 오차 제곱으로 계산되는 MAE, MSE는 함수 사용시 실제값, 예측값 위치가 바뀌어도 됨 -> 비추 **
** MAPE, R^2는 실제값, 예측값 순서가 바뀌면 안됨 **

Actual Value

Accuracy (정확도) : TN + TP / TN + FP + FN + TP
Precision (정밀도) : TP / FP + TP
Recall (재현율) : TP / FN + TP

TN : True Nagative : 음성으로 잘 예측한 것 : 음성을 음성이라고 예측
FP : False Positive : 양성으로 잘 못 예측한 것 : 음성을 양성이라고 예측
FN : False Negative : 음성으로 잘 못 예측한 것 : 양성을 음성이라고 예측
TP: True Positive : 양성으로 잘 예측한 것 : 양성을 양성이라 예측

#Accuracy
- 정분류율 이라고 부르기도 함
- 전체 중에서 Positive 와 Negative 로 정확히 예측한 (TN + TP) 비율
- TN로 예측한 경우도 옳은 예측임을 고려
- 가장 직관적으로 모델 성능을 확인가능

#Precision
- FP + TP 중에서 TP인 비율 ( 비가 내릴 것으로 예측한 날 중에서 실제 비가 내린 날의 비율 / 암이라 예측한 환자 중에서 실제 암인 환자의 비율)
- 정밀도가 낮을 경우 발생하는 상황 ( 비가 오지 않는데 비가 온다고 했으니 불필요한 우산을 챙기는 수고 발생 / 암이 아닌데 암이라 했으니 불필요한 치료 발생)

#Recall
- FN + TP 중에서 TP 비율
- 민감도(Sensitivity)라고 부르는 경우가 많음
 ( 실제 비가 내린 날 중에서 비가 내릴 것으로 예측한 날의 비율 / 실제 암인 환자 중에서 암이라고 예측한 환자의 비율 )
- 재현율이 낮을 경우 발생하는 문제 ( 비가 내리는 날 내리지 않을 것이라 했으니 우산을 챙기지 않아 비를 맞음 / 암인 사람에게 암이 아니라 했으니 심각한 결과 초래 )

#Specificity
- TN + FP 중 TN 비율 ( 실제 비가 내리지 않은 날 중에서 비가 내리지 않을 것으로 예측한 날의 비율 / 실제 암이 아닌 환자 중에서 암이 아니라고 예측한 환자의 비율 )
- 특이도가 낮을 경우 발생하는 문제 ( 비가 오지 않는데 비가 온다고 했으니 불필요한 우산을 챙기는 수고 발생 / 암이 아닌데 암이라 했으니 불필요한 치료 발생 )

#F1-Score
- 정밀도와 재현율의 조화평균
- 분자가 같지만 분모가 다를 경우 조화평균이 정확
- 정밀도와 재현율이 적절하게 요구 될 때 사용

F1 - Score = 2 * ( 1/ (1/Precesion + 1/Recall) ) = (2 * Precesion * Recall) / (Precesion + Recall)
